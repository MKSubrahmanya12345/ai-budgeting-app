## рџЏѓвЂЌв™‚пёЏ Getting Started

### Prerequisites
- Node.js (v18+)
- npm / yarn
- Firebase Account
- Google Gemini API Key

### Backend Installation
1. `cd backend`
2. `npm install`
3. Create a `.env` file with:
   ```env
   GEMINI_API_KEY=your_key
   PORT=5000
   ```
4. `npm run dev`

### Frontend Installation
1. `cd frontend`
2. `npm install`
3. `npm run dev`
